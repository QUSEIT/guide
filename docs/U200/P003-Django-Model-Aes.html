Django models AES 加密、解密¶

思路¶

django Field 在存入 Mysql 时会调用 get_prep_value 方法，从 Mysql 中取数据时会调用 to_python 方法，根据这个逻辑，我们就有方案啦。

  1. 存数据时加密（重写 get_prep_value方法）;
  2. 取数据时解密（重写 to_python 方法）。

安装¶

AES 包安装

pip install pycrypto
